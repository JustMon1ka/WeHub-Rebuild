### 项目简介

#### Auth模块

auth模块实现了用户的注册、登录、登出功能。用户可以通过注册账号来创建个人资料，并使用登录功能访问论坛的其他功能。登出功能确保用户在完成操作后能够安全退出系统。

并且通过token实现了用户的身份验证和会话管理，确保用户在访问受保护的资源时具有适当的权限。以及持久化存储用户的会话信息，确保用户在关闭浏览器或重新登录后仍然保持登录状态。

还通过SMTP协议实现了邮件发送功能，用于用户注册时的邮箱验证和密码重置等操作。

#### User模块

user模块负责管理用户的个人信息和设置。用户可以查看和编辑自己的个人资料，包括用户名、头像、简介等。用户还可以通过该模块查看他人的个人资料和动态，以及关注列表和粉丝列表。

实现了在粉丝列表的懒加载，以及关注和取消关注功能，用户可以在用户头像上悬浮显示用户的详细信息，以及点击头像跳转到用户的个人主页。

#### Circle模块
Circle模块负责圈子（社区）的全流程管理与互动，包括圈子的创建、编辑与删除，分类检索与详情查看，以及成员的申请加入、审批、退出与移除等。模块同时支持圈内活动管理（创建、更新、删除）与成员积分、排行榜等互动玩法，并提供圈子头像、横幅与活动封面等媒体上传能力。所有后端写操作与统一JWT认证衔接。列表、分类、详情、成员公开查询等接口允许匿名访问，确保发现与浏览体验，已登录用户则可进行加入、审批、活动参与与奖励领取等操作。
